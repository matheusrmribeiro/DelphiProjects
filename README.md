## VCLBorderlessForm
**What I learned?**
* Inheritance
* Create a borderless form without losing common Windows commands

```
WIN + ↑ (Align the form Client)
WIN + ↓ (Minimize the form)
WIN + →(Align the form Right)
WIN + ←(Align the form Left)
```
